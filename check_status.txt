Run these commands on your server:

# 1. Check if bot service is running
sudo systemctl status shirzadbot

# 2. Check if port 5010 is listening
netstat -tlnp | grep 5010

# 3. Try to connect to the bot directly
curl http://localhost:5010

# 4. Check recent logs
tail -100 /var/www/shirzadBot/logs/app.log

# 5. Check for errors
tail -100 /var/www/shirzadBot/logs/error.log

# 6. View systemd logs
sudo journalctl -u shirzadbot -n 50 --no-pager
